# Requirements
Before setting up the project you need to make sure these tools are installed on your environment:
 - Docker
 - Docker Compose
 - Make

## Getting Started
To set up the project run below commands:
```shell
make setup
make migrate
```

## Running Tests
Run below command to run tests:
```shell
make test
```
